EJERCICIO 2: INTEGRA 7 ELEMENTOS HTML

Botones

Son útiles para realizar acciones específicas, como enviar un formulario, ejecutar scripts, o interactuar con el usuario.

Inputs

Se utiliza para crear controles interactivos dentro de formularios, como campos de texto, botones de opción, casillas de verificación, entre otros. Es un elemento versátil que admite varios tipos de entrada

Area de texto

se utiliza para crear un área de texto multilínea, donde el usuario puede ingresar más de una línea de texto. Es comúnmente utilizado en formularios cuando se necesita permitir a los usuarios escribir respuestas largas, comentarios o descripciones.

Iconos

No son un "elemento HTML" en sí, sino que son representados a través de etiquetas como <"i">, <"span">, o <"svg">, y su visualización es controlada mediante bibliotecas de íconos como Font Awesome o Material Icons.

Check boxes

Es un elemento que permite a los usuarios seleccionar una o más opciones en un formulario.

Selectores

Se utiliza para crear menús desplegables en formularios, permitiendo que los usuarios elijan entre varias opciones.

Rangos

Se utiliza para crear un control deslizante (slider) que permite a los usuarios seleccionar un valor dentro de un rango especificado. Este tipo de entrada es útil cuando deseas que el usuario elija un valor numérico dentro de un intervalo de forma interactiva, como el ajuste de volumen, brillo o un valor de puntaje.

Listas

Permiten estructurar contenido en forma de puntos o pasos numerados, lo que resulta útil para organizar información de manera clara.
