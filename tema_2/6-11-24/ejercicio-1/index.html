EJERCICIO 1: ELEMENTOS, ATRIBUTOS Y CLASES HTML

Elementos

Los elementos son las estructuras básicas de HTML que definen el contenido y la organización de una página web. Generalmente, están compuestos por una etiqueta de apertura, contenido (opcional) y una etiqueta de cierre.

Atributos

Los atributos son propiedades adicionales que proporcionan información sobre un elemento. Se especifican dentro de la etiqueta de apertura y usualmente están compuestos por un nombre y un valor.

Clases

Las clases son atributos que se usan para agrupar elementos en categorías para aplicarles estilos o comportamientos específicos a través de CSS o JavaScript. Se definen con el atributo class.
